Subiectele pentru aceasta sesiune a concursului sunt:

  1. Subiect 1

    Se citește n număr natural. Calculați suma numerelor naturale mai mici sau egale cu n.

  2. Subiect 2

    Se dau n numere întregi. Calculaţi cel mai mic dintre cele n numere date.

  3. Subiect 3

    Să se scrie un program care citește un șir de n numere naturale şi determină valoarea maximă din șir și de câte ori apare.
